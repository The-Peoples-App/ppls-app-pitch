## Published Slide Deck
View the People's App Slide Deck online.

**Key-Commands/Navigation**:
- Arrow Keys <kbd>&larr;</kbd> <kbd>&rarr;</kbd> to Navigate
- `s` for Speaker View (Timer, Upcoming Slides, Notes, Pacing)
- `?` to list all commands/controls
